# Habit Tracker

A simple habit tracker application built with Python using OOP principles and Streamlit for the UI.
This project was created as a practice project.

---

## Project Idea
The application helps users build consistency by tracking daily habits.
Users can check in every day and maintain streaks based on consecutive habit completion.

---

## Features
- Add new habits
- Delete existing habits
- Daily habit check-in
- Streak calculation for consecutive days
- Best habit & worst habit analysis
- Success rate for each habit

---

## Built With
- Python
- Streamlit
- Object-Oriented Programming (OOP)
